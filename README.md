##Portfolio
This repository contains the files and code for my portfolio website. This website showcases my skills and projects. It is designed to provide a comprehensive overview of my professional profile.

## Features

- Home section provides brief information about myself.
- About section provides brief description about me.
- Education section shows year of education like SSC,HSC with descriptions .
- Contact section allowing visitors to get in touch with me with the help of links.
- Resume is attached at the bottom which provides more information about me and technical skills.
